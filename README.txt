CONTENU DU DOSSIER
- index.html : page principale du calendrier
- styles.css : mise en forme
- script.js : logique du calendrier
- vercel.json : configuration simple pour Vercel

DEPLOIEMENT RAPIDE
1. Créer un dépôt GitHub vide
2. Uploader ces fichiers à la racine du dépôt
3. Importer le dépôt dans Vercel
4. Déployer
5. Le site aura automatiquement une adresse en .vercel.app
