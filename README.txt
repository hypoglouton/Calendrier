CALENDRIER V3 - GITHUB + VERCEL

Contenu
- application React prête pour Vercel
- calendrier mensuel format A4
- bibliothèque de vignettes à glisser-déposer
- contraste de couleurs renforcé
- clic sur une vignette pour modifier heure, titre, note
- suppression d'une vignette
- sauvegarde automatique en local dans le navigateur

Mise en ligne la plus simple
1. Dézipper le dossier
2. Envoyer tous les fichiers sur GitHub
3. Importer le repo dans Vercel
4. Cliquer sur Deploy

Commandes locales si besoin
- npm install
- npm run dev

Fichiers principaux
- src/App.jsx
- src/main.jsx
- package.json
- vite.config.js
