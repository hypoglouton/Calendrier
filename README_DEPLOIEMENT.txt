Déploiement Vercel / GitHub

IMPORTANT
Les fichiers doivent être à la RACINE du projet, pas dans un sous-dossier.

Fichiers à garder :
- index.html
- styles.css
- app.js
- vercel.json

Procédure :
1. Supprimer les anciens fichiers du repo si besoin.
2. Envoyer ces 4 fichiers directement à la racine.
3. Redéployer.
4. Faire Ctrl + F5.

Cette version est 100 % statique : aucun build Vite/React n'est nécessaire.
